Texto y encabezados

Contenido

En HTML, las palabras clave de algunas etiquetas son obvias, pero hay algunas etiquetas que usan una forma abreviada. Por ejemplo, usamos p para un parráfo.

Codificar un párrafo es muy similar a codificar un botón. Necesita un etiqueta de apertura, seguida de su texto y una etiqueta de cierre.

Podemos crear encabezados en HTML para enfatizar ciertos textos. Hay 6 tamaños de encabezados en HTML, de h1 a h6.

La etiqueta de encabezado enfatiza los textos haciéndolos más grandes y más audaces. h1 es el más grande y más audaz y h6 es el mas pequeño y menos audaz.

Preguntas

¿Cuál de estas etiquetas HTML crea un párrafo?

  • <p>
  • <button>
  • <video>
  • <h1>

¿Cuál de estas etiquetas HTML es un encabezado?

  • <p>
  • <h1>
  • <button>
  • <video>

¿Cuál de estas etiquetas HTML crea un encabezado más pequeño que <h1>?

  • <p>
  • <h6>
  • <button>
  • <video>

¿Cuál de estas etiquetas HTML crea el encabezado más pequeño?

  • <h1>
  • <h6>
  • <h4>
  • <h3>
