Etiquetas HTML

El código HTML les dice a los navegadores web cómo mostrar contenido como imágenes, enlaces y texto en una página web.

HTML es un lenguaje de marcado de computadora que estructura todas las páginas web en internet.

¡Ya aprendimos lo suficiente como para responder nuestra primera pregunta del cuestionario!

En HTML, las cosas giran en torno a las etiquetas. Las etiquetas comienzan con <, seguidas de una palabra clave y terminan con >.

¡Creamos nuestra primera etiqueta <button> en el archivo index.html!

<button> es una etiqueta de apertura. Le dice al navegador web que queremos comenzar a mostrar un botón.

La mayoria de las etiquetas en HTML vienen en pares, tienen una etiqueta de apertura y una de cierre. Lo que codifico anteriormente es la etiqueta de apertura de un elemento botón.

Vamos a codificar la etiqueta de cierre del botón agregando </button> después de la etiqueta del botón de apertura.
