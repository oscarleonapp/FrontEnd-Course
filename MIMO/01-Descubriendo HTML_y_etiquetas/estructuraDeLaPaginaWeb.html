Estructura de la página web

Contenido

Un elemento es lo que llamamos la etiqueta de apertura, la etiqueta de cierre y el código intermedio.

Para crear una página web, necesitará una etiqueta HTML de apertura y cierre.

Los elementos html y body forman la estructura de una página web. Colocamos los elementos que queremos mostrar dentro del cuerpo.

Más adelante, aprenderemos sobre otros elementos que van dentro de <html></html> para dar estilo a las páginas web y hacerlas interactivas.

Debemos colocar contenidos, como encabezados, dentro de las etiquetas del cuerpo.

Cuestionario

¿Cuál de estos es un elemento?

  • <h1>
  • <button>Hello, World!</button>
  • <p>
  • <h1>

¿Cuál de estas opciones creará una página web HTML vacía?

  • <h1></h1>
  • <html></html>
  • <button></button>

¿Cuál de estos códigos define el cuerpo de una página web?

  • <h1></h1>
  • <body></body>
  • <html></html>
